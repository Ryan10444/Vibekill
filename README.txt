Vibekill is a first person shooter game that focuses on timing shots and abilities to the beat of music. The accuracy of the timing will effect the effectiveness of the bullet or ability. The game will be an online multiplayer shooter, where players can select a weapon that has a distinct song and BPM.

How to commit:
"feat: (New large features)"
"fix: (Bug fixes)"
"chore: (Optimization/Maintenance)"
"change: (Changes to game that are not considered a new feature)"
"refactor: (Changes to code structure that does not change functionality)"
"docs: (Changes to documents/text files that do not have an effect on the game"
